# Discord Bot Builder

This tool allows you to generate a custom Discord bot executable in just a few steps.

## Instructions

### 1. Disable Your Antivirus
Before running the tool, temporarily disable your antivirus software. Some antivirus programs may falsely detect the builder as suspicious and prevent it from working correctly.

### 2. Run the Tool
Launch the application by double-clicking the executable.

### 3. Enter Your Discord Bot Token
When prompted, paste your Discord bot token into the input field.

### 4. Build the Executable
After entering the token, the tool will automatically generate the bot executable (`.exe`).

### 5. Done
Once the build process is complete, your generated executable will be available in the output folder.

## Requirements

- Windows
- A valid Discord Bot Token
- Internet connection (if required by the builder)

## Disclaimer

Use this software responsibly and only with Discord bots that you own or are authorized to manage.
